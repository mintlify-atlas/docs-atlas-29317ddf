SST makes it easy to build modern full-stack applications on AWS, Cloudflare, and Vercel using TypeScript, with live development and type-safe infrastructure.

## Quick Start

Get up and running with SST in minutes:

<Steps>
  <Step title="Install SST">
    <CodeGroup>
    ```bash npm
    npm install sst
    ```
    ```bash yarn
    yarn add sst
    ```
    ```bash pnpm
    pnpm add sst
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Initialize your project">
    ```bash
    npx sst init
    ```
  </Step>
  
  <Step title="Start developing">
    ```bash
    npx sst dev
    ```
  </Step>
</Steps>
