The `Bucket` component lets you add an [AWS S3 Bucket](https://aws.amazon.com/s3/) to your app.

## Constructor

```ts title="sst.config.ts"
const bucket = new sst.aws.Bucket("MyBucket");
```

## Parameters

### access

<ParamField path="access" type='"public" | "cloudfront"'>
  Enable public read access for all files in the bucket.

  ```ts
  {
    access: "public"
  }
  ```

  <Tip>
    If using a `Router` to serve files, use `"cloudfront"` access instead.

</ParamField>

## Methods

### notify

<Method>
  Subscribe to events from this bucket with functions, queues, or topics.

  ```ts
  bucket.notify({
    notifications: [
      {
        name: "MySubscriber",
        function: "src/subscriber.handler"
      }
    ]
  });
  ```

  Subscribe with a queue:

  ```ts
  const queue = new sst.aws.Queue("MyQueue");

  bucket.notify({
    notifications: [
      {
        name: "MySubscriber",
        queue: queue,
        events: ["s3:ObjectCreated:*"]
      }
    ]
  });
  ```
</Method>

## Examples

### Enable public access

```ts title="sst.config.ts"
new sst.aws.Bucket("MyBucket", {
  access: "public"
});
```

### Add notifications

```ts title="sst.config.ts"
const bucket = new sst.aws.Bucket("MyBucket");

bucket.notify({
  notifications: [
    {
      name: "MySubscriber",
      function: "src/subscriber.handler",
      events: ["s3:ObjectCreated:*"]
    }
  ]
});
```

### Link to a function

```ts title="sst.config.ts"
const bucket = new sst.aws.Bucket("MyBucket");

new sst.aws.Function("MyFunction", {
  handler: "src/api.handler",
  link: [bucket]
});
```
